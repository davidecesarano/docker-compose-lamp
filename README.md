# Docker-compose LAMP
This is a boilerplate for LAMP based projects that run on Docker-Compose. 
The main services included are:
* Ubuntu 18.04
* Apache 2
* MariaDB
* PHP 7.4
* Node.js
* Composer

## Usage
```
$ git clone https://github.com/davidecesarano/docker-compose-lamp.git
$ cd docker-compose-lamp
$ docker-compose up -d
```
